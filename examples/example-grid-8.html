Przyklad grid 8

Przykład użycia grid na wzór flex za pomocą auto-fill lub auto-fit 

/* Przykład 1 */
.grid-container {
display:grid; 
grid-template-columns: repeat(auto-fill, 10rem);
justify-content: center;
}
 
/* Przykład 2 */
.grid-container {
  display:grid; 
  grid-template-columns: repeat(auto-fit, 10rem);
  justify-content: center;
}
