Przyklad flex 6

Przykład ułożenia poszczególnych elementów za pomocą align-self

.flex-container {
 display: flex;
 gap: 10px;
 justify-content: center;
 align-items: center;
}
          
          
.flex-container div:nth-child(2){
 width: 30px;
 height: auto;
 align-self: stretch;
}
          
.flex-container div:nth-child(3) {
 width: 80px;
 height: 40px;
 align-self: flex-start;
}
          
.flex-container div:nth-child(4) {
 width: 100px;
 height: 120px;
}
          
.flex-container div:nth-child(5) {
 width: 40px;
 align-self: flex-end;
}
