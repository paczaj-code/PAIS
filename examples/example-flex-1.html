Przyklad flex 1

Prosty przykład z użyciem flex. W przeciwieństwie do grid domyślnie elementy zajmują jak najmniej miejsca w kontenerze.

.flex-container {
 display: flex;
 gap: 0px;
}
            
.flex-container div:nth-child(2) {
 width: 30px;
}

.flex-container div:nth-child(3) {
 width: 80px;
}
            
.flex-container div:nth-child(4) {
 width: 100px;
}
            
.flex-container div:nth-child(5) {
 width: 40px;
}
