Przyklad flex 5

Przykład ułożenia wszystkich elementów za pomocą justify-content i align-items

/* Przykład 1 */
.flex-container {
 display: flex;
 gap: 10px;
justify-content: center;
}

/* Przykład 2 */
.flex-container {
    display: flex;
    gap: 10px;
    align-items: center;;
}         
        
          
.flex-container div:nth-child(2) {
 width: 30px;
 height: 100px;
}
          
.flex-container div:nth-child(3) {
 width: 80px;
 height: 40px;
}
          
.flex-container div:nth-child(4) {
 width: 100px;
 height: 120px;
}
          
.flex-container div:nth-child(5) {
width: 40px;
}
