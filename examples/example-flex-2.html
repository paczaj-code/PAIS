Przyklad flex 2

flex-grow pozwala elementom zajmować wolne miejsce w kontenerze. 

.flex-container {
 display: flex;
 gap: 3px;
}

/* Przykład 1  rozłożenne wszystkich elemntów*/
flex-container > div {
 flex-grow: 1;
}

/* Przykład 2  rozłożenne nieparzystych elemntów*/
.flex-container div:nth-child(odd) {
 flex-grow: 1;
}
