Przyklad grid 4

Przykład z użyciem nazw obszarów.

.grid-container {
 display:grid; 
 grid-template-areas: "element1 element1 element1 ."
                          "element3 element4 element4 element2"
                          "element5 element5 element5 .";
}

.grid-container div:nth-child(1) {
 grid-area: element1;
}

.grid-container div:nth-child(2) {
 grid-area: element2;
}

.grid-container div:nth-child(3) {
 grid-area: element3
}

.grid-container div:nth-child(4) {
 grid-area: element4
}

.grid-container div:nth-child(5) {
 grid-area: element5
}
