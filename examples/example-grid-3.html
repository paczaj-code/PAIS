Przyklad grid 3

Przykład z uzyciem named lines. Nazwy podajemy w nawiasch kwadratowych

Składnia dla

grid-template-<columns or rows> :[fist-line-name] <size> [name1 name2] <size> .... [last-line-name];

.grid-container {
    display:grid; 
    grid-template-columns: [column-medium-start] 2fr [column-medium-end column-thin-start] 1fr [column-thin-end column-wide-start] 3fr [column-wide-end];
    grid-template-rows: [row-medium-start] 2fr [row-medium-end row-wide-start] 4fr [row-wide-end row-thin-start] 1fr [row-thin-end]
}

.grid-container div:nth-child(1) {
    grid-row: row-medium-start / row-wide-end;
    grid-column: column-medium-start / column-thin-end;
}
